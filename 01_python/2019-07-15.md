간단한 명령어

ctrl + enter = 현재 셀 실행

shift + enter = 실행 + 다음셀 선택(다음 셀 없으면 새로운 셀 생성)

alt + enter = 실행 + 다음 셀 생성

d + d = 셀 삭제

a = 현재 셀 위에 새로운 셀 생성

b = 현재 셀 아래에 새로운 셀 생성

shift = 다중 선택
